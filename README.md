# Simple blockchain and mining in Rust


## create block in blockchain
```bash
curl -X POST http://localhost:8080 \
     -H "Content-Type: application/json" \
     -d '{"bpm":21}'
```

## get blockchain
```bash
curl localhost:8080 | jq
# or
curl localhost:8080 
```
